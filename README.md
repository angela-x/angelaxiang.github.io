## Customization

1. **Personal Information**: Edit `index.html` to update:
   - Name and title in the hero section
   - About section content
   - Experience entries
   - Education details
   - Skills
   - Projects
   - Contact links (email, LinkedIn, GitHub)

2. **Styling**: Modify `styles.css` to change:
   - Colors (update CSS variables in `:root`)
   - Fonts
   - Spacing and layout
   - Animations

3. **Functionality**: Update `script.js` for:
   - Additional interactive features
   - Custom animations
   - Form handling (if adding a contact form)

## Deployment

This site is ready for GitHub Pages. Simply:

1. Push your changes to the `main` branch
2. Go to your repository settings
3. Navigate to Pages
4. Select the `main` branch as the source
5. Your site will be available at `https://angelaxiang.github.io`
